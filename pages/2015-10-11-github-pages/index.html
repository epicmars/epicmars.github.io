Github Pages博客搭建

参考： Using Jekyll with Pages

建立博客仓库

参考Github Pages搭建博客Git仓库，clone到本地即可。

安装Ruby

根据ruby官网的方法：

sudo apt-get install ruby-full

安装后发现是1.9.3版本，jekyll需要ruby最低版本为2.0，使用brightbox ppa源进行安装：

sudo apt-get install python-software-properties
sudo apt-add-repository ppa:brightbox/ruby-ng
sudo apt-get update
sudo apt-get install ruby2.2 ruby2.2-dev

另一种使用RVM安装Ruby的方法见http://rvm.io/：

gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3
sudo apt-get install curl
\curl -sSL https://get.rvm.io | bash -s stable autolibs=enabled --ruby --rails --trace

设置Ruby版本

rvm use ruby-2.2.1 -default
rvm use default

安装Bundler

sudo gem install bundler

发现存在网络连接的问题，可以使用国内镜像https://ruby.taobao.org/

gem sources --remove https://rubygems.org/
gem sources -a https://ruby.taobao.org/
gem sources -l

安装Jekyll并运行

在站点仓库的根目录下新建文件Gemfile，添加如下内容：

source 'https://rubygems.org'
gem 'github-pages'

然后进行安装：

bundle install

生成博客站点：

jekyll new myblog

最后测试安装是否成功：

bundle exec jekyll serve

发现未安装JavaScript运行时环境，安装Nodejs，可以下载编译好的软件包，修改~/.bash_aliases文件将bin目录添加的环境变量。
