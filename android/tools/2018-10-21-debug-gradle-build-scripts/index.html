Android Studio之Gradle的使用

调试Gradle脚本

  1. 添加Remote运行Configuration
  2. 在命令行运行gradle任务，并添加参数-Dorg.gradle.daemon=false -Dorg.gradle.debug=true -Dorg.gradle.jvmargs=’-Xdebug -Xrunjdwp:transport=dt_socket,server=y,address=5005’

调试Annotation Processor

与调试Gradle脚本类似，如果需要支持Kotlin，那么可以先关闭kapt的配置，将注解处理器配置为annotationProcessor依赖进行调试。

设置Gradle代理

在项目根目录的gradle.properties中进行如下配置：

systemProp.http.proxyHost=127.0.0.1
systemProp.http.proxyPort=8888
systemProp.http.proxyUser=userid
systemProp.http.proxyPassword=password
systemProp.http.nonProxyHosts=*.nonproxyrepos.com|localhost

systemProp.https.proxyHost=127.0.0.1
systemProp.https.proxyPort=8888
systemProp.https.proxyUser=userid
systemProp.https.proxyPassword=password
systemProp.https.nonProxyHosts=*.nonproxyrepos.com|localhost
