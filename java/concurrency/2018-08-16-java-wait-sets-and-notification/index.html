Java多线程：等待集与通知（Wait Sets and Notification）

每个对象除了有一个关联的监视器，还有一个关联的等待集合。一个等待集合是一个线程的集合。

一个线程的中断状态和Thread类中处理中断的类方法会影响等待集的操作，此外，Thread类中的用于sleeping线程以及joining其它线程的类方法所具有的性质也是从等待和通知操作得来的。

等待

等待操作发生在对wait()以及定时形式的wait(long millisecs)，wait(long millisecs, int nanosecs)的调用。

通知

通知发生在对notify和notifyAll方法的调用。

中断

中断发生在对Thread.interrupt以及用于依次对其调用的方法的调用，例如ThreadGroup.interrupt。
