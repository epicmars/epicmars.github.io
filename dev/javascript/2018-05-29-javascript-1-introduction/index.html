JavaScript: 简介

1. JavaScript简介

虽然JavaScript和ECMAScript常常表达同样的含义，但JavaScript的含义比ECMA-262中规定的要多得多。一个完整的JavaScript实现应该由下列三个不同部分组成。 - 核心（ECMAScript） - 文档对象模型（DOM） 文档对象模型（DOM，Document Object Model）是针对XML但经过扩展用于HTML的API。DOM把整个页面映射为一个多层节点结构。HTML或XML页面中的每个组成部分都是某种类型的节点，这些节点又包含不同的数据类型。 - 浏览器对象模型（BOM） 提供访问和操作浏览器窗口的方法和接口，开发人员使用BOM可以控制浏览器显示的页面以外的部分。BOM作为JavaScript实现的一部分但还没有相关标准。

2. 在HTML中使用JavaScript

<script>元素

解释器对<script>元素内部的代码或外部JavaScript文件中的代码时，页面的处理会暂时停止，页面中的其余内容都不会被浏览器加载或显示。

按照惯例，所有<script>元素都应该放在页面的<head>元素中，但这以为着必须等<head>中包含的所有JavaScript文件都被下载、解释和执行完成以后，才能开始呈现页面内容（浏览器在遇到<body>标签时才开始呈现内容）。为了避免这一问题，现代Web应用程序一般把全部JavaScript引用放在<body>元素中页面的内容后面，如下所示：

<!DOCTYPE html>
<html>
    <head>
        <title>Example HTML Page</title>
    </head>
    <body>
        <!-- 这里放内容 -->
        <script type="text/javascript" src="example1.js"></script>
        <script type="text/javascript" src="example2.js"></script>
    </body>
</html>
