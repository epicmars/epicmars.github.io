使用文档特征进行正文提取

要提取的正文是body元素本身或者其子元素所包含的内容，它作为一个块级元素，具有如下特征：

  • 链接密度较小，即包含较少的链接
  • 文本密度大，将标签数量相比较，包含较多文字
  • 由于作为正文组成部分的段落也具有上述两种特征，因此正文必须是符合上述要求的最大的块（也可能是多个块组成）
