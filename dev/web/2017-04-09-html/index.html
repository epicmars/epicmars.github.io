HTML

HTML可以给一段文字赋予不同的意义（段落，列表还是表格），也可以用于组织文档结构，还可以将图片和视频等内容嵌入到页面中。

块级元素与行内元素

  • 块级元素 块级元素形成了一个页面上的可见区域，总是位于新的一行，其后的元素也会位于新的一行。块级元素不会嵌套在行内元素中，但可以嵌套在另一个块级元素中。

  • 行内元素 行内元素位于块级元素内，并且只包含文档中的小块内容，不是整个段落和一组内容。

注意:HTML5重新定义了元素类别。

元素属性值用双引号或单引号都可以，可以将一个嵌套在另一个中使用。

HTML中的多个空格会缩减为一个。

HTML头

HTML头不会展示在页面中，它包含了页面的元数据，例如页面标题，CSS文件链接，页面favicon图标等等。

应用CSS与JavaScript

使用<link>和<script>元素可以分别将CSS和JavaScript应用到HTML中。

  • <link>总是位于文档的头中。
  • <script>不必位于文档头中，常常将其放到文档body底部，在关闭标签</body>的前面。

HTML文本

为什么我们需要语义？

我们周围的任何地方都依赖于语义，我们依赖于以往的经验来告诉我们日常对象的功用，当我们见到什么时，我们就知道它的功能是方面。例如，我们知道红灯停绿灯行，如果有什么地方是反过来的就很奇怪了。

创建超链接

网络之所以成为网络超链接起到重要作用，使得一个文档可以链接到另一个文档或其它资源。

可以链接到一个HTML文档的某一特定部分，称之为文档片段。

绝对与相对URL

相对URL：相对当前页面的链接

链接最佳实践

  • 使用清晰简洁的链接文案
  • 尽可能使用相对链接，如果使用绝对URL，浏览器会从查找服务器的真实位置开始，也就是说它会请求DNS进行域名解析，然后从服务器找到所需的文件。
  • 链接到非HTML资源时，留下清晰的指示
  • 链接到下载时使用download属性，可以用于提供默认保存文件名。

文本格式化

表达计算机代码

  • <code>：用于标记代码块
  • <pre>: 用于标记宽度固定的文本块，其中空格会保留下来。
  • <var>: 用于标记变量名
  • <kbd>: 用于标记键盘输入
  • <samp>: 用于标记计算机输出

无语义的结构标签

有时不能找到一个理想的元素将一些项目组织在一起或包装某些内容。有时你可能仅仅想将一组元素分为一组以便应用作为某个CSS样式或JavaScript脚本的单一入口。对于这种情况，HTML提供了<div>和<span>元素。
