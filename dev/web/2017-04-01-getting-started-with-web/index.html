开始Web之旅

安装必要的工具

一个文本编辑器，这里推荐Brackets，它提供了页面的实时预览与代码提示的功能。还有一个现代浏览器Firefox, Chrome都是不错的选择。

好了现在就可以进入web开发的大门了，作为有经验的开发者，你需要的工具可能还有版本控制系统Git，自动化工具Grunt或者Gulp

HTML

HTML是一种组织网页结构并呈现网页内容的标记语言。它包含一系列的元素，元素由标签和内容组成，元素还可以拥有属性。

没有内容的元素称为空元素。

CSS

CSS是为网页指定样式的代码，与HTML一样，它不是一门编程语言，而是样式表语言，可以让你选择性地为HTML中的元素指定样式。

CSS规则集剖析

也简称为CSS规则，如下图所示：

选择器：规则集开头的元素名。

定义：包括属性和属性值，为元素的某一属性指定样式。

每个规则集位于一对花括弧内，每个定义由分号分开，定义内属性和属性值由冒号分开。

一个规则集可以指定多种类型的选择器，并用逗号分开，这样可以将统一规则集应用到不同的元素。

不同类型的选择器

  • 元素选择器：选择指定类型的所有HTML元素
  • ID选择器：选择特定ID的元素，在一个给定的HTML页面中，元素的id不能重复，即每个ID只能允许指定一个元素
  • 类选择器：选择特定类型的元素，一个给定页面中，可以选中多个具有相同类型的元素
  • 属性选择器：选择页面中具有指定属性的元素
  • 伪类选择器：选择指定状态下的特定元素

还有其它类型的选择器，可以参考选择器指南

盒模型

CSS布局主要基于盒模型，每个在页面上占据空间的区块都有以下属性：

  • 内边距：围绕内容的空间
  • 边界：内边距外的实线
  • 外边距：元素外部的空间

JavaScript

JavaScript是一个为网站添加交互性的编程语言，当应用于一个HTML文档时，可以实现许多事情。先从简单的特性开始，它可以实现跑马灯、相册、动态布局、按钮点击事件响应。在有更多经验后，可以用于制作游戏，2D动画和3D图形，以及数据库驱动的综合性应用等等。

JavaScript自身相当简洁，也非常具有弹性。开发者已经在JavaScript核心语言上编写了大量的工具，让我们可以轻松地获取到大量额外的功能，这些包括：

  • 浏览器API，构建在浏览器中的API提供了类似动态生成HTML和设置CSS样式的功能，摄取并操作用户摄像头的视频流，或者生成3D图形与声音采样。
  • 第三方API可以让开发者将其它网站提供的内容集成到自己的站点。
  • 第三方框架和库可以让你迅速创建站点和应用。

数据类型

  • 字符串：用引号包围的文本
  • 数值：一个数字
  • 布尔值：ture和false
  • 数组：一个可以存储多个值的结构
  • 对象：基本上JavaScript中所有值都是对象，并且都可以保存到一个对象中。
