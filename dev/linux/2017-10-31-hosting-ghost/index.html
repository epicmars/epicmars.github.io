Ghost服务器搭建

官方安装指南

For installing on production, we officially recommend the following stack:

Ubuntu 16.04
MySQL
NGINX (minimum of 1.9.5 for SSL)
Systemd
Node v6 installed via NodeSource
At least 1GB memory (swap can be used)
A non-root user for running ghost commands

安装Ghost

参考Ghost官方指南,安装NodeJs，之后为了加快安装速度，可以安装cnpm:

$ sudo npm install -g cnpm --registry=https://registry.npm.taobao.org

安装指南中的进程与启动管理只能在ubuntu16中使用，用的aliyun服务器还是14.04的，需要配置Upstart脚本：

description "ghost service for my website"

start on runlevel [2345]
stop on runlevel [!2345]

setuid ghost
setgid ghost

respawn

env NODE_ENV=production
chdir /var/www/ghost
exec /usr/bin/node /usr/bin/ghost run
