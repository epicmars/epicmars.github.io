Ajax

Ajax(Asynchronous JavaScript + XML)本身并不是一种技术，而是由Jesse James Garrett于2005年提出的一个方法，将已存在的多种web技术进行整合，使得Web应用可以在无需重载整个浏览器页面的情况下，快速、增量地更新用户界面，这使得应用响应更加迅速。

什么是AJAX

总而言之，它是使用XMLHttpRequest对象与服务器进行通信。它可以接收和发送各种格式的信息，包括JSON，XML，HTML，甚至文本文件。
