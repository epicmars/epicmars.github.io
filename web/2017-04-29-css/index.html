CSS

CSS是如何工作的

当一个浏览器展示一个文档，它必须将文档的内容与样式信息结合。它分两步处理文档：

  1. 浏览器将HTML和CSS转换为DOM（Document Object Model）。DOM代表了计算机内存中的文档，它将文档的内容和样式结合在一起。
  2. 浏览器展示DOM的内容。

选择器

简单选择器

根据元素类型或者其class和id匹配一个或多个元素。

  • 元素（类型）选择器
  • 类选择器：包含一个点号，后跟一个类名，一个类名是HTML的class属性的一个不包含空格的任意值，需要注意的是一个文档中多个元素可以有相同的的类，单个元素可以有多个由空格分隔的类。
  • ID选择器：由已#号跟一个元素的id名组成，任意元素都可以有一个唯一的id名，它是id属性的值。
  • 通配选择器：它是最大的王牌（ultimate joker）,它可以选择文档中的所有元素。

属性选择器

属性选择器会根据属性和属性值匹配相应的元素，一般语法由一对方括弧([])及其所包围的属性名和可选的用于匹配满足一定条件的属性值所组成。

伪类选择器

伪类选择器并不选择元素，而是选择元素的某一部分或仅仅在某种上下文中的元素，它有两种主要的类型——伪类和伪元素。

伪类

一个CSS伪类是一个关键词，跟随在一个冒号(:)后，并且添加到选择器的尾部，表明你仅想在被选择的元素处于特定的状态时对其进行样式化。

伪元素

伪元素非常类似伪类，但它们有一些不同点。它们是跟随在双冒号(::)后的关键词，可以加入到选择器的尾部来选择一个元素的特定部分。

组合选择器和选择器组

组合选择器可以将多个选择器结合到一起，允许你选择其它元素中的内部或相邻元素，它有如下四种类型：

  • 选择器组，- , -，选择以逗号分隔的多个选择器中人一个相匹配的元素，即将样式应用到所有与列出类型相匹配的元素。
  • 后代选择器，-（space) -，允许你选择其它元素中的一个嵌套元素（不必是直接子元素）。
  • 子元素选择器，- > - ，允许你选择一个元素的直接子元素。
  • 相邻兄弟选择器，- + - , 允许你选择一个元素的相邻直接相邻的元素（例如，右边直接相邻的元素，并且处于同一层级）。
  • 通用相邻兄弟选择器，- ~ -， 允许你选择一个元素的任意相邻元素（例如处于同一层级，但不必直接相邻）。

CSS值和单位

CSS中有许多值的类型，主要包括：

  • 数字值：长度值，如元素宽度，边框粗细或者字体大小。无单位的整数，如相对直线宽度或者运行一个动画的次数。
  • 百分数：也可以用于指定尺寸或长度——例如相对于父容器的长度和宽度，或者指定默认font-size。
  • 颜色：用于指定背景颜色，文本颜色等等。
  • 坐标位置：例如指定一个位置元素相对于屏幕左上角的位置。
  • 函数：例如指定背景图片或者背景图片渐变。

数字值

  • 长度和尺寸

    • px: 像素被作为绝对单位，因为无论其它相关的设置它们总是具有相同的尺寸。其它绝对尺寸有：
      • mm, cm, in：毫米，厘米或英寸
      • pt, pc：点(1/72英寸)或picas（12点）
    • em：1em和当前元素的font-size一样（更具体一点，就是大写字母M的宽度。在应用CSS样式之前，默认网页的基本font-size为16px。但需要注意字体大小是从父元素继承来的，一旦父元素设置了不同的字体大小，与em等价的像素值就会变的复杂。em是你将在web开发中最常使用到的相对单位。
    • ex, ch：它们分别是小写字母x的高度和数字0的宽度。
    • rem：即root em，其工作原理和em基本一样，除了它总是等于默认基本font-size；继承字体大小没有影响，这听起来像是一个比em更好的选择，但rem不能在老版本的IE浏览其中起作用。
    • vw, vh：分别是viewport宽度的1/100和高度的1/100。再次，它们和rem一样并没有得到广泛的支持。

  • 无单位的值 在CSS中，你有时会遇到无单位的值。

    • 无单位的行距
    • 动画次数

百分数

绝大多数使用数字值的地方都可以使用百分数。

颜色

在CSS中有许多指定颜色的方式。

  • 关键词
  • 16进制值
  • RGB
  • HSL
  • RAGB与HSLA
  • 透明度

函数

在编程中函数是一个可以被复用的代码块，可以被执行多次来完成重复性的工作。在CSS中也有函数。例如指定颜色的rgb()和hsl()等。

background-color: rgba(255,0,0,0.5);
background-color: hsla(240,100%,50%,0.5);

也有其它许多函数，例如：

/* calculate the new position of an element after it has been rotated by 45 degress */
transform: rotate(45deg);
/* calculate the new position of an element after it has been moved across 50px and down 60px */
transform: translate(50px, 60px);
/* calculate the computed value of 90% of the current width minus 15px */
width: calc(90%-15px);
/* fetch an image from the network to be used as a background image */
background-image: url('myimage.png');
